# 二字棋：再创新天

一款以中国古代建筑智慧为灵感的原创剧情驱动策略博弈游戏。项目以“守御（框）”与“攻伐（棋）”为核心机制，将榫卯结构、客家土楼、都江堰、故宫中轴线等古代建筑原理转化为可交互的攻守规则，构建出兼具文化表达、策略深度与剧情推进的独立游戏体验。

## 项目简介

《二字棋：再创新天》是一款基于 Python 开发的单机策略 + 剧情向独立游戏。玩家在对局中分别扮演守御方或攻伐方，通过放置“框”与“棋”展开非对称博弈。游戏包含三章主线、共三十关剧情关卡，并提供“灵境”和“冥想”两种训练模式。玩家在推进过程中会积累“天气”“地气”“元宝”等资源，并在最终结局中导向不同分支。

### 基础规则

1. 开局选择身份：守御方或攻伐方。
2. 守御方先落“框”，框为一个 2×2 的守势区域。
3. 攻伐方随后在该守势内部连续落两枚“棋”。
4. 从第二次落框开始，新的框至少要有一条边与已有框相连。
5. 双方按规则循环对抗共五轮。
6. 若在第五轮结束前攻伐方形成五星连珠，则攻伐方获胜；否则守御方获胜。

### 特色机制

- **非对称博弈**：不是传统“棋子 VS 棋子”，而是“框 VS 棋”的攻守对抗。
- **双落子攻势**：攻伐方每回合一次性落两枚棋子，强化组合与爆发感。
- **文化规则映射**：将榫卯、土楼、都江堰、中轴线等建筑智慧转化为玩法逻辑。
- **多结局系统**：根据“天气 / 地气”累计结果，触发不同哲学结局。
- **特殊关卡设计**：包含搭桥、突破封锁、3D 立方体棋盘等变化玩法。

## 游戏内容

### 三章剧情

- **第一章：生死江湖**
  - 师父授道、主角下山、遇渔夫、闯山贼。
  - 第十关为桥梁搭建类特殊关卡。

- **第二章：街巷阡陌**
  - 协助行商、街巷逃生、潜入佛殿、逼近皇城。
  - 第二十关要求以更强攻势突破层层封锁。

- **第三章：飘摇皇庭**
  - 进入皇庭，介入天子与叛军首领之间的最终对决。
  - 第三十关采用立体 3D 棋盘与补天石主题机制。

### 训练模式

- **灵境**：自由对战练习模式。
- **冥想**：由玩家同时承担攻守两方，用于推演与自我训练。

### 成长与奖励

- **元宝**：用于在“家乡”场景中购买框与棋的新皮肤。
- **天气 / 地气**：影响最终剧情走向。
- **家乡系统**：整合训练模式与商铺系统。

## 技术栈

- **Python 3**
- **Pygame**：图形渲染、音频播放、事件交互
- **OpenCV**：视频帧读取与格式转换
- **JSON**：游戏存档与配置持久化
- **自研 3D 数学库**：实现最终关立方体棋盘的旋转、投影、判定与交互

## 核心实现

### 模块设计

项目采用模块化架构，核心能力包括：

- 棋盘抽象
- 动画系统
- 存档系统
- AI 系统
- 剧情系统
- 3D 渲染
- 家乡整合

### 关键技术点

- 使用 `resource_path()` 兼容开发环境与 PyInstaller 打包环境的资源路径。
- 使用 `VideoLoop` 统一管理背景视频与动画循环。
- 使用 `AssetLoader` 统一加载图片、音频、视频资源。
- 使用 JSON 保存进度、天气、地气、元宝与皮肤解锁状态。
- 第 30 关通过纯数学方式完成立方体棋盘渲染，并将立体五连珠判断降维为二维检测。

## 项目亮点

- **玩法原创性强**：独立构建“守御 / 攻伐”双层规则。
- **文化表达明确**：将中国古代建筑智慧直接嵌入玩法与叙事。
- **轻量技术路线**：未依赖重型游戏引擎，仍实现了 2D + 3D 混合体验。
- **剧情与玩法融合**：教学、成长、叙事与玩法迭代同步推进。
- **易上手，难精通**：基础规则直观，但高阶对局强调远期布局与攻防转化。

## 运行说明

### 环境要求

建议使用：

- Python 3.10 及以上
- 已安装 `pygame`
- 已安装 `opencv-python`

### 安装依赖

```bash
pip install pygame opencv-python
```

### 启动方式

请将项目主程序文件与全部资源文件放在同一工程目录下，然后运行你的游戏入口脚本，例如：

```bash
python <入口脚本>.py
```

> 注意：这里的 `<入口脚本>.py` 需要替换为你项目中的实际主程序文件名。

项目依赖较多媒体资源，包括但不限于：

运行前请确认这些资源文件与代码中的相对路径保持一致。
